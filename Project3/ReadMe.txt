Contents of this Zip file

1. ReadMe.txt - this document
2. Prog3.txt - A demonstration executable of programming assignment 3.  The .exe
	extension has been changed to .txt to get past some firewalls.  Change the
	.txt to .exe to run.  Double click the .exe file to run.
3. Book.h - the header file defining the Book data structure.  You may add this to your
	project and use it as is.
4. HWBookDB.txt - A text file containing all data needed to build the Library binary tree.
5. SprintReport_P3.docx - Scrum report for sprints 1 and 2 for this assignment.  Fill
	this in and return it with your source code.
